offer_001 (整数除法)

Method一

  • 先考虑边界条件
  • 采用原始的方法，求a/b，则迭代地求a=a-b;
  • 结果res每次增加一
  • 需要注意临界值，即res超过最大值的时候

Method二

  • 同样先考虑边界条件
  • 方法一res每次值只变化一次，速度过慢。
  • 求a中能最多包括多少个b，由于整数限制原因，无论a，b取何值（变为正数后），a中最多包含230个b(b=1时)
  • 此外，若用b*230>a来进行判断有可能产生溢出的情况，因此采用a/230<b来进行判断
  • 此外，本题不允许使用/和*，因此用位运算符<<和>>进行运算。
