    1. 函数外面的语句必须要以关键字开头（例如 var，const，func等等）

  • 重要Golang中，变量一定要在声明后才能使用，并且Go语言的变量声明后一定要使用

  • 关于Printf和Println函数，Printf用于需要格式化输出的时候，而Println不能输出格式化内容。

  • Go中bool默认值为false，并且Go中不允许将整型强制转换为布尔型

  • Go中的转义字符

    转义符 含义
    \r 回车符（返回行首）
    \n 换行符（直接跳到下一行同列位置）
    \t 制表符
    \'  单引号
    \" 双引号
    \\ 反斜杠

  • Go语言的字符有两种：

    1. unit8类型，或者叫byte型，代表一个ASCII码字符

    2. rune类型，代表一个UTF-8字符 实际上是一个int32类型 用于处理中文，日文又或者其他符合字符时

    3. 字符串底层是一个byte数组，所以可以和[]byte类型互相转换，字符串时不能修改的，字符串由byte字节组成，因此字符串长度时byte字节的长度，rune类型可以用来表示utf8字符，一个rune字符由一个或者多个byte组成。

    4. 字符串的修改:要求改字符串，需要先将字符串转换为[]rune或者[]byte，完成后再转换为string,如下：

    5. ```go
      func changeString() {
      s1 := “big”
      // 强制类型转换
      byteS1 := []byte(s1)
      byteS1[0] = ‘p’
      fmt.Println(string(byteS1))

      s2 := “白萝卜”
      runeS2 := []rune(s2)
      runeS2[0] = ‘红’
      fmt.Println(string(runeS2))
